# template-svelte

Vite + Svelte + SvelteKit + TailwindCSS

## Get Started

Click **Use Template** or this [link](https://github.com/yzITI/template-svelte/generate).

Change project name in  `package.json`
```
"name": "project-name"
```

Install dependency

```
npm i
```

Run dev server

```
npm run dev
```
